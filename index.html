Admin Refill System


Having learnt C++ for in my spare time, as the first language I learnt, I created a hypothetical admin system for a refill centre. The admin can see all information in one go, calculate the cost of an item based off the weight, check the stock levels, and add the item to a basket (still a work in progress). When working on this project, the hardest part was using a libary in C++, as well as a class. I am still working on objectors, which would allow me to use a constructor overload to update each item.
